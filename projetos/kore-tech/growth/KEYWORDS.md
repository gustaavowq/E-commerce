## 5. Anchor text interno (links cruzados)

- **De blog → landing/PDP:** sempre que mencionar componente, link com texto = nome do componente (não "clique aqui"). Ex: "uma boa opção pra esse uso é a [RTX 4070 Super](/produtos/rtx-4070-super)".
- **De landing persona → PCs prontos:** "Veja [3 PCs prontos pra esse uso](/pcs?persona=valorant-240fps)".
- **De landing persona → builder:** "Quer customizar? Use o [PC Builder](/montar?persona=valorant-240fps) — vem filtrado pra esse perfil".
- **De PDP componente → builder:** "Use no [PC Builder](/montar?inicial=ryzen-7-7700x) — adicionamos automaticamente".
- **De PDP componente → blog:** "Não sabe se essa CPU rende pro seu jogo? Veja [como escolher CPU pra gamer](/blog/qual-cpu-2026)".
